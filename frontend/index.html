Interactive REST API Reference Documentation

LeadGen AI provides a powerful, modular REST API. Below are the key endpoints available for managing leads, triggering scraping jobs, starting AI analysis, and tracking email outreach.

Leads Management API

GET /api/leads — List and filter collected leads
Query Parameters: status, city, category, source, min_score, search, page, page_size.
GET /api/leads/{id} — Fetch details & outreach history for a lead
PATCH /api/leads/{id} — Update status, email, or notes
Payload example: 
{
  "status": "interested",
  "email": "contact@business.com",
  "notes": "Spoke on the phone, requested proposal."
}
DELETE /api/leads/{id} — Permanently delete a lead

AI Intelligence API

POST /api/leads/{id}/analyze — Triggers background website checks, custom scoring, and Gemini AI analysis
POST /api/leads/{id}/generate-messages — Generates custom outreach pitches, WhatsApp messages, and multi-step follow-ups using Gemini

Scraper Engines API

POST /api/scrape/google-maps — Start Google Maps local crawler (extracts ratings, SSL, emails, speed metrics)
POST /api/scrape/upwork — Trigger Upwork job scraper based on keywords & budgets
POST /api/scrape/[freelancer | linkedin | fiverr] — Triggers Freelancer projects, LinkedIn company pages, or Fiverr demand scrapers

Outreach & Tracking API

POST /api/outreach/send — Delivers email, registers open pixel, and generates uuid redirect urls
GET /api/track/open/{tracking_id} — Serves 1x1 tracking pixel and increments open_count
GET /api/track/click/{tracking_id}?url=... — Logs link clicks and redirects lead securely to encoded destination url
